createCanvas

Esta función crea el lienzo del canvas

createCanvas(idContainer: string, idCanvas: string, width: number, height: number, border: string, tagContainerCanvas: string, appendToContainer: string): element
Parameters
idContainer (string) del contenedor
idCanvas (string) id del canvas
width (number) ancho del canvas
height (number) altura del canvas
border (string) expresión css para el borde del canvas
tagContainerCanvas (string) etiqueta html del container del container
appendToContainer (string) elemento donde se añade el canvas
Returns
element: el contexto del canvas

horizontalLines

Esta función crea varias líneas horizontales

horizontalLines(xstart: number, ystart: number, xend: number, yend: number, width: number, color: string, cap: string, ctx: string, newYStart: number, newYend: any, numDeLineas: number, newYEnd: number): void
Parameters
xstart (number) punto donde comienza en el eje x
ystart (number) punto donde comienza en el eje y
xend (number) punto donde termina en el eje x
yend (number) punto donde termina en el eje y
width (number) ancho de la línea
color (string) color de la línea
cap (string) borde de la línea puede ser "butt", "round", "square"
ctx (string) contexto del canvas
newYStart (number) nuevo valor para el punto de comienzo de la nueva línea en el eje y
newYend (any)
numDeLineas (number) número de líneas a dibujar
newYEnd (number) nuevo valor para el punto de fin de la nueva línea en el eje y
Returns
void: no retorna nada
